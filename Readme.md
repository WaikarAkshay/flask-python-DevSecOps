# Akshay Waikar - Personal Portfolio

Minimalist single-page portfolio website showcasing my profile as a **DevOps Engineer** based in Pune, India.  
Built with clean HTML + CSS and served using a lightweight **Flask** application.

Current vibe: professional, infrastructure-minded, terminal-inspired accents, subtle hover effects.


## Features

- Single-page layout (fits most desktop screens without scrolling)
- Responsive design (mobile becomes scrollable naturally)
- Subtle modern interactions (hover lifts, glows, gradient accents)
- DevOps engineer aesthetic: monospace headings, teal pipeline-like highlights
- Served via Flask for easy extensibility (contact form, metrics, future API, etc.)

## Tech Stack

- **Frontend**: HTML5, CSS3 (vanilla, no frameworks), Google Fonts (Inter + JetBrains Mono)
- **Backend**: Python + Flask (minimal static file serving)
- **Deployment-ready**: Works with Gunicorn, Render, Railway, Fly.io, Heroku, etc.
